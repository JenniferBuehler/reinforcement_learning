Detailed Description

template<class State, typename Value = float>
class rl::Reward< State, Value >

Implements the reward function. This can be a function or a lookup in a table.

Parameters
Statetemplate class for State description. Prerequisite: Must support < operator, an be uniquely identifiable, i.e. suitable to use in a std::map as key. Even if the Reward is being calculated as a function by the underlying implementation, it is to be made sure that a map-lookup can be used as well for each state.
Valuethe value type to be used as reward
