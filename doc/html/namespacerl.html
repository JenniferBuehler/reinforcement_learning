class  LearningController
 Interface to learning algorithms. After setting the initial state of the environment for the learner, it can be used to update the state each time step. Internally, each state will lead to a recommended action by the learner, which in turn can be used to transfer the system into a new state. This will have to be done outside this class, e.g. in a subclass of Domain, which in turn uses the transition function to transfer between states. More...
